# KindScript

**KindScript** is a free, open-source toolkit that helps people write clear, compassionate letters when they’re facing medical or support-related barriers — like insurance denials, prescription costs, or hard-to-navigate systems.

💛 No logins.  
🔒 No data tracking.  
📄 Just gentle tools to help you say what matters.

---

## 🌱 What You Can Do Here

- Generate ready-to-edit letters or emails for things like:
  - Appealing denied medications
  - Asking your doctor for alternatives or samples
  - Requesting help without shame

- Get plain-language guidance on:
  - How to ask for help
  - What “prior authorization” really means
  - Scripts for hard conversations around care
